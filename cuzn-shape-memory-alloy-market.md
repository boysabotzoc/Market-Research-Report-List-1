<p><ul><li>One-Way Memory Effect</li><li>Two-Way Memory Effect</li></ul></p>
<p><p>The CuZn shape memory alloy market is characterized by two main types based on memory effects: one-way and two-way. The one-way memory effect allows the material to return to a predetermined shape upon heating after being deformed, making it useful in applications like actuators and medical devices. In contrast, the two-way memory effect enables the alloy to alternate between two shapes with temperature changes, enhancing its versatility in applications such as robotics and smart textiles, where active shape changing is crucial.</p></p>
